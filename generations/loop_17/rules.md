# Loop 17 Cellular Automata Rules: Parasitic Memetic Infections & Mind Control

This document contains 10 distinct, mathematically rigorous cellular automata (CA) rules focused on the domain of **Parasitic Memetic Infections & Mind Control**. 

These rules model cognitive contagion, host hijacking, neural network rewiring, memetic mutational drift, immunological subversion, sleeper agent networks, polarizing echo chambers, dopaminergic hijacking, mutualistic subversion, and tipping-point herd conformity. Each rule is designed for a 1D lattice where the state of cell $x$ at discrete time step $t$ is denoted by $S_t(x) \in \{0, 1, \dots, N-1\}$. The space-time patterns generated by these rules visually demonstrate how complex parasitic dynamics spread, hijack, and self-organize within cognitive networks.

## Rule Definitions

### Rule 1: Cognitive Contagion (Basic Epidemic Memetics)
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Susceptible Mind ($S$, resting cognitive state).
  * State $1$: Latent Infection / Incubation ($E$, idea is planted but dormant).
  * State $2$: Active Propagation / Spreader ($I$, actively transmitting the meme).
  * State $3$: Subverted Node / Permanent Beacon ($V$, hijacked state, continuously propagates).
  * State $4$: Temporarily Resistant ($R$, cognitive immunity phase, decays over time).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
  Let $C_I = \mathbb{I}(S_t(x-1) = 2) + \mathbb{I}(S_t(x+1) = 2)$ be the count of active spreaders.  
  Let $C_V = \mathbb{I}(S_t(x-1) = 3) + \mathbb{I}(S_t(x+1) = 3)$ be the count of subverted beacons.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  $$S_{t+1}(x) = \begin{cases}
  1 & \text{if } s = 0 \text{ and } (C_V \ge 1 \text{ or } C_I = 2) \\
  0 & \text{if } s = 0 \text{ and not } (C_V \ge 1 \text{ or } C_I = 2) \\
  2 & \text{if } s = 1 \text{ and } (C_I + C_V \ge 1) \\
  0 & \text{if } s = 1 \text{ and } C_I + C_V = 0 \\
  3 & \text{if } s = 2 \text{ and } C_I \ge 1 \text{ and } C_V \ge 1 \\
  4 & \text{if } s = 2 \text{ and not } (C_I \ge 1 \text{ and } C_V \ge 1) \\
  3 & \text{if } s = 3 \\
  0 & \text{if } s = 4 \text{ and } C_V = 0 \\
  4 & \text{if } s = 4 \text{ and } C_V \ge 1
  \end{cases}$$
* **Mathematical/Physical Rationale:** Models the transition from initial exposure to full-scale cognitive capture. Susceptible minds require either close contact with a permanent beacon (3) or multiple active spreaders (2) to initiate incubation (1). Once incubated, if reinforcing exposure continues, the host becomes active (2). If exposed to high-density feedback (both a spreader and a beacon nearby), the node undergoes a phase transition to a permanent beacon (3). Resistance (4) is temporary and decays unless kept active by nearby subverted beacons.
* **Expected Visual Behavior:** Clean, propagating wavefronts of infection (state 2) followed by bands of resistance (state 4). Permanent beacons (state 3) appear as solid vertical columns that periodically spawn nested, outwards-propagating V-shaped waves of infection.

### Rule 2: Host Hijacking (Toxoplasmosis Memetica)
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Healthy Host ($H$, resting, susceptible).
  * State $1$: Latent Infection ($L$, host carrying the memetic parasite, asymptomatic).
  * State $2$: Hijacked Host ($J$, zombified state, exhibits vector-seeking movement).
  * State $3$: Vector / Memetic Source ($V$, stationary source/predator that consumes hijacked hosts).
  * State $4$: Spent / Devoured ($X$, refractory recovery state).
* **Neighborhood ($N(x)$):** Radius $r=2$ including the cell itself:
  $$N(x) = \{x-2, x-1, x, x+1, x+2\}$$
  Let $V_L = \mathbb{I}(S_t(x-1) = 3) + \mathbb{I}(S_t(x-2) = 3)$ indicate a vector is to the left.  
  Let $V_R = \mathbb{I}(S_t(x+1) = 3) + \mathbb{I}(S_t(x+2) = 3)$ indicate a vector is to the right.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } S_t(x-1) = 3 \text{ or } S_t(x+1) = 3 \\
    2 & \text{else if } (S_t(x-1) = 2 \text{ and } S_t(x-2) \neq 3 \text{ and } V_R > 0) \\
      & \text{or } (S_t(x+1) = 2 \text{ and } S_t(x+2) \neq 3 \text{ and } V_L > 0) \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } V_L + V_R > 0 \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    4 & \text{if } S_t(x-1) = 3 \text{ or } S_t(x+1) = 3 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 3$$
  - If $s = 4$:
    $$S_{t+1}(x) = 0$$
* **Mathematical/Physical Rationale:** Inspired by host-parasite biology (e.g., *Toxoplasma gondii* altering rodent behaviors to attract them to feline vectors). In a 1D lattice, this is mathematically framed as a directed state-transport mechanism. Hijacked hosts (2) shift their position (propagate) towards the nearest vector (3), while healthy hosts (0) remain still. When a hijacked host reaches a cell adjacent to a vector, it is consumed and enters the spent state (4), which decays back to healthy (0), while the vector (3) remains nourished and stationary.
* **Expected Visual Behavior:** Stationary vertical columns of vectors (state 3). Healthy cells near the vectors turn into state 1 (latent), then convert to state 2 (hijacked) and form clear diagonal tracks converging onto the state 3 columns. Upon arrival, they produce a brief single-pixel flash of state 4 before resetting.

### Rule 3: Synaptic Plasticity Subversion
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Quiescent Synapse ($Q$, resting neural pathway).
  * State $1$: Right-moving Signal ($S_{\rightarrow}$, forward signal propagation).
  * State $2$: Left-moving Signal ($S_{\leftarrow}$, backward signal propagation).
  * State $3$: Parasitic Center ($P$, static source of rewiring agent).
  * State $4$: Subverted Synapse ($W$, permanent channel routing all signals toward the parasite).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } S_t(x) \neq 4 \text{ and } S_t(x-1) = 1 \\
    2 & \text{else if } S_t(x) \neq 4 \text{ and } S_t(x+1) = 2 \\
    4 & \text{else if } S_t(x-1) = 4 \text{ and } S_t(x+1) = 3 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    4 & \text{if } S_t(x+1) = 3 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    4 & \text{if } S_t(x-1) = 3 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 3$$
  - If $s = 4$:
    $$S_{t+1}(x) = 4$$
* **Mathematical/Physical Rationale:** Models the rewiring of neural networks by parasitic memetic triggers. Under normal conditions, signals propagate freely through quiescent synapses (0). When a signal (1 or 2) hits the parasitic center (3), it triggers local synaptic plasticity subversion. The synapse is permanently rewired (4) to redirect all incoming traffic towards the parasitic center, starving the rest of the neural network of communication.
* **Expected Visual Behavior:** Signals appear as thin diagonal lines propagating left (state 2) and right (state 1). When they hit the static vertical column of the parasitic center (state 3), they freeze, turning into subverted synapses (state 4). Over time, this creates an expanding wedge of state 4 centered around state 3, which absorbs all incoming signal lines.

### Rule 4: Viral Memetic Drift and Strain Competition
* **States ($N$):** $6$ states ($0, 1, 2, 3, 4, 5$).
  * State $0$: Susceptible Host ($S$, uninfected mind).
  * State $1$: Strain $A$ ($I_A$, highly contagious, transient).
  * State $2$: Strain $B$ ($I_B$, moderately contagious, persistent).
  * State $3$: Strain $C$ ($I_C$, highly virulent mutated strain, bypasses immunity).
  * State $4$: Co-immune State ($R_{AB}$, resistant to Strains A and B).
  * State $5$: Exhausted Host ($E$, refractory/depleted state).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
  Let $N_k = \mathbb{I}(S_t(x-1) = k) + \mathbb{I}(S_t(x+1) = k)$ for $k \in \{1, 2, 3\}$.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_3 \ge 1 \text{ or } (N_1 \ge 1 \text{ and } N_2 \ge 1) \\
    1 & \text{else if } N_1 \ge 1 \\
    2 & \text{else if } N_2 \ge 1 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = 4$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_1 \ge 1 \\
    5 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 5$$
  - If $s = 4$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_3 \ge 1 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 5$:
    $$S_{t+1}(x) = 0$$
* **Mathematical/Physical Rationale:** Models the evolutionary drift of competing viral memes. Strains A and B propagate through the susceptible population, but where their fronts collide, the conceptual friction causes a mutation event, generating Strain C. Strain C represents a radicalized variant that bypasses the co-immune state (4) generated by prior exposure to A or B. Strain B is highly persistent and drains the host, leading to exhaustion (5), while Strain A clears rapidly.
* **Expected Visual Behavior:** Clean diagonal paths of Strain A (1) and Strain B (2) propagating across the grid. At the points where A and B collide, a new, fast-moving wave of Strain C (3) emerges, leaving behind wide bands of exhausted hosts (5) that slowly recover to state 0.

### Rule 5: Immunological Subversion (Antibody Hijacking)
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Vulnerable Host ($V$, healthy, unprotected).
  * State $1$: Infected Spreader ($I$, active meme carrier).
  * State $2$: Immune Defender ($D$, active cognitive defense).
  * State $3$: Trojan Horse ($T$, hijacked defender, highly infectious).
  * State $4$: Immunized Host ($M$, resistant to standard spreaders).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
  Let $N_I = \mathbb{I}(S_t(x-1) = 1) + \mathbb{I}(S_t(x+1) = 1)$.  
  Let $N_D = \mathbb{I}(S_t(x-1) = 2) + \mathbb{I}(S_t(x+1) = 2)$.  
  Let $N_T = \mathbb{I}(S_t(x-1) = 3) + \mathbb{I}(S_t(x+1) = 3)$.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } N_T \ge 1 \\
    1 & \text{else if } N_I \ge 1 \text{ and } N_D = 0 \\
    4 & \text{else if } N_D \ge 1 \text{ and } N_I = 0 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    4 & \text{if } N_D \ge 1 \text{ and } N_I = 0 \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_I \ge 2 \\
    2 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 3$$
  - If $s = 4$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } N_T \ge 2 \\
    4 & \text{otherwise}
    \end{cases}$$
* **Mathematical/Physical Rationale:** Models the subversion of host defense mechanisms. Cognitive immune defenders (2) normally sweep the lattice, clearing infected spreaders (1) and leaving behind immunized hosts (4). However, if an immune defender is surrounded by infection ($N_I \ge 2$), it becomes subverted into a Trojan Horse (3). Trojan Horses are highly contagious, permanently active, and can penetrate immunizations (4) when multiple Trojan Horses coordinate.
* **Expected Visual Behavior:** Moving fronts of defenders (state 2) pushing back infected regions (state 1). Where the defense lines are breached and defenders are surrounded, blocks of state 3 (Trojan Horse) form, spawning rapid, devastating infection waves that break through previously established immunized barriers (state 4).

### Rule 6: Sleeper Agent Activation (Broadcast Subversion)
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Healthy Host ($H$, susceptible).
  * State $1$: Sleeper Agent ($S_{dormant}$, carrying dormant infection).
  * State $2$: Broadcast Trigger Wave ($B$, moving signal).
  * State $3$: Activated Agent ($S_{active}$, actively subverting).
  * State $4$: Spent / Burned-out ($X$, refractory state).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
  Let $N_{active} = \mathbb{I}(S_t(x-1) = 3) + \mathbb{I}(S_t(x+1) = 3)$.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } N_{active} \ge 1 \\
    2 & \text{else if } S_t(x-1) = 2 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } S_t(x-1) = 2 \text{ or } S_t(x+1) = 2 \\
    3 & \text{else if } N_{active} \ge 1 \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = 0$$
  - If $s = 3$:
    $$S_{t+1}(x) = 4$$
  - If $s = 4$:
    $$S_{t+1}(x) = 0$$
* **Mathematical/Physical Rationale:** Models latent networks. Sleeper agents (1) remain completely dormant, behaving like empty/healthy spaces until a broadcast signal (2) sweeps across the lattice. The broadcast signal is a solitary wave. When it hits a sleeper (1), the agent activates (3), infecting adjacent healthy hosts into new sleepers (1) before burning out (4).
* **Expected Visual Behavior:** A single, sharp diagonal line representing the broadcast wave (state 2) sweeps through the grid. In its wake, invisible sleepers (state 1) light up as active nodes (state 3) which quickly expand into small V-shaped patterns of new sleepers before turning off (state 4).

### Rule 7: Algorithmic Polarizing Echo Chambers
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Moderate / Neutral ($M$, unpolarized).
  * State $1$: Left-leaning Moderate ($L$).
  * State $2$: Right-leaning Moderate ($R$).
  * State $3$: Extremist Left ($E_L$, polarized/infected).
  * State $4$: Extremist Right ($E_R$, polarized/infected).
* **Neighborhood ($N(x)$):** Radius $r=2$ including the cell itself:
  $$N(x) = \{x-2, x-1, x, x+1, x+2\}$$
  Let $W_L = \sum_{d \in \{-2,-1,1,2\}} (\mathbb{I}(S_t(x+d) = 1) + 2\mathbb{I}(S_t(x+d) = 3))$ be the total left-wing weight.  
  Let $W_R = \sum_{d \in \{-2,-1,1,2\}} (\mathbb{I}(S_t(x+d) = 2) + 2\mathbb{I}(S_t(x+d) = 4))$ be the total right-wing weight.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } W_L > W_R + 1 \\
    2 & \text{else if } W_R > W_L + 1 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } W_L > W_R + 2 \\
    0 & \text{else if } W_R > W_L \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    4 & \text{if } W_R > W_L + 2 \\
    0 & \text{else if } W_L > W_R \\
    2 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } W_R > W_L + 4 \\
    3 & \text{otherwise}
    \end{cases}$$
  - If $s = 4$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } W_L > W_R + 4 \\
    4 & \text{otherwise}
    \end{cases}$$
* **Mathematical/Physical Rationale:** Models the polarization of beliefs under reinforcing algorithmic feed mechanisms (confirmation bias). Extremist states (3 and 4) exert double the persuasive weight of moderates (1 and 2). Cells shift their states toward the dominant ideological alignment of their neighbors. Extremists are highly stable, requiring a massive concentration of opposing views to even return to a moderate state, creating strong resistance to depolarization.
* **Expected Visual Behavior:** Spontaneous phase separation. A randomly initialized lattice rapidly self-sorts into large, solid, homogeneous blocks of state 3 (Extremist Left) and state 4 (Extremist Right), separated by thin, flickering boundary walls of moderates (1, 2) and neutrals (0).

### Rule 8: Dopaminergic Hijacking (Neuromodulation)
* **States ($N$):** $5$ states ($0, 1, 2, 3, 4$).
  * State $0$: Basal State ($B$, resting host).
  * State $1$: Parasite Spore ($P$, short-lived infectious agent).
  * State $2$: Infected Host ($I$, carrying the parasite, seeking excitation).
  * State $3$: Dopamine Spike / Hyperactive Spreader ($H$, high activity).
  * State $4$: Desensitized / Refractory ($D$, receptor burnout state).
* **Neighborhood ($N(x)$):** Radius $r=1$ including the cell itself:
  $$N(x) = \{x-1, x, x+1\}$$
  Let $N_P = \mathbb{I}(S_t(x-1) = 1) + \mathbb{I}(S_t(x+1) = 1)$.  
  Let $N_H = \mathbb{I}(S_t(x-1) = 3) + \mathbb{I}(S_t(x+1) = 3)$.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } N_P \ge 1 \\
    1 & \text{else if } N_H \ge 1 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = 0$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_H \ge 1 \text{ or } S_t(x-1) = 2 \text{ or } S_t(x+1) = 2 \\
    2 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 4$$
  - If $s = 4$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } N_P \ge 2 \\
    0 & \text{otherwise}
    \end{cases}$$
* **Mathematical/Physical Rationale:** Models memetic hijacking of the brain's reward centers. Spores (1) infect basal cells (0). The infected cells (2) remain dormant until stimulated by other infected carriers or hyperactive spreaders, triggering a Dopamine Spike (3). The spike lasts one step, during which they release a spore (1) and then crash into a receptor-depleted refractory state (4). Re-infection of a depleted host is only possible if exposed to a high spore density ($N_P \ge 2$).
* **Expected Visual Behavior:** Explosive, localized bursts of state 3 (flashing pixels) that instantly die out into state 4, leaving behind a dispersing cloud of spores (state 1) that jump to neighboring carrier cells. This creates a highly dynamic, crackling wave propagation structure resembling localized neural seizures.

### Rule 9: Symbiotic Subversion (Mutualistic Hijacking)
* **States ($N$):** $6$ states ($0, 1, 2, 3, 4, 5$).
  * State $0$: Vulnerable Host ($V$, resting state).
  * State $1$: Symbiont ($S$, beneficial state, protects against parasites).
  * State $2$: Parasitic Infection ($P$, destructive state).
  * State $3$: Subversive Intermediate ($I$, density-triggered mutation).
  * State $4$: Fully Subverted Host ($H$, resource-drained).
  * State $5$: Necrotic Void ($X$, dead host cell).
* **Neighborhood ($N(x)$):** Radius $r=2$ including the cell itself:
  $$N(x) = \{x-2, x-1, x, x+1, x+2\}$$
  Let $N_S = \sum_{d \in \{-2,-1,1,2\}} \mathbb{I}(S_t(x+d) = 1)$ be the count of adjacent symbionts.  
  Let $N_P = \sum_{d \in \{-2,-1,1,2\}} \mathbb{I}(S_t(x+d) = 2)$ be the count of adjacent parasites.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } N_P \ge 2 \text{ and } N_S < 1 \\
    1 & \text{else if } N_S \ge 1 \text{ and } N_P = 0 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_S \ge 3 \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } N_S \ge 2 \\
    5 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 4$$
  - If $s = 4$:
    $$S_{t+1}(x) = 5$$
  - If $s = 5$:
    $$S_{t+1}(x) = \begin{cases}
    0 & \text{if } N_S \ge 1 \\
    5 & \text{otherwise}
    \end{cases}$$
* **Mathematical/Physical Rationale:** Models mutualistic subversion where a meme initially presents as a beneficial symbiont. Symbionts (1) protect hosts against parasites (2) and heal necrotic voids (5) back to healthy states (0). However, if the local density of symbionts exceeds a critical threshold ($N_S \ge 3$), the symbiont mutates into a subversive agent (3). This agent quickly takes over the host (4) and consumes its resources, leaving behind a necrotic void (5).
* **Expected Visual Behavior:** Stable, growing islands of symbionts (state 1) that shield the host from parasites (state 2). Once an island grows too large, its center turns yellow (state 3) and collapses into a necrotic black void (state 5), creating a cyclic breathing pattern of expansion and internal decay.

### Rule 10: Herd Subversion and Tipping Points (Bootstrap Memetics)
* **States ($N$):** $4$ states ($0, 1, 2, 3$).
  * State $0$: Independent Mind ($I$, rational agent).
  * State $1$: Skeptical Agent ($S$, questioning status quo).
  * State $2$: Infected / Subverted Agent ($A$, conformed).
  * State $3$: Dogmatic Enforcer ($D$, permanently subverted, exerts double influence).
* **Neighborhood ($N(x)$):** Radius $r=2$ including the cell itself:
  $$N(x) = \{x-2, x-1, x, x+1, x+2\}$$
  Let $N_A = \sum_{d \in \{-2,-1,1,2\}} \mathbb{I}(S_t(x+d) = 2)$ be the count of subverted agents.  
  Let $N_D = \sum_{d \in \{-2,-1,1,2\}} \mathbb{I}(S_t(x+d) = 3)$ be the count of dogmatic enforcers.  
  Let $N_{total} = N_A + 2N_D$ be the weighted social influence.
* **Transition Function:**
  Let $s = S_t(x)$ be the current state of cell $x$.
  - If $s = 0$:
    $$S_{t+1}(x) = \begin{cases}
    1 & \text{if } N_{total} \ge 3 \\
    0 & \text{otherwise}
    \end{cases}$$
  - If $s = 1$:
    $$S_{t+1}(x) = \begin{cases}
    2 & \text{if } N_{total} \ge 4 \\
    0 & \text{else if } N_{total} \le 1 \\
    1 & \text{otherwise}
    \end{cases}$$
  - If $s = 2$:
    $$S_{t+1}(x) = \begin{cases}
    3 & \text{if } N_D \ge 2 \text{ or } (N_A \ge 3 \text{ and } N_D \ge 1) \\
    1 & \text{else if } N_{total} = 0 \\
    2 & \text{otherwise}
    \end{cases}$$
  - If $s = 3$:
    $$S_{t+1}(x) = 3$$
* **Mathematical/Physical Rationale:** Models social tipping points (bootstrap percolation). Belief adoption is driven by weighted local consensus. Dogmatic enforcers (3) have twice the influence of subverted agents (2). An independent agent (0) becomes skeptical (1) under moderate pressure, and fully conforms (2) under high pressure. Once conformed, if surrounded by dogmatics, the host is permanently locked into a dogmatic state (3). Only complete isolation ($N_{total}=0$) allows a subverted host to recover.
* **Expected Visual Behavior:** Sharp boundaries between rational domains (state 0) and dogmatic domains (state 3). Dogmatic structures expand outward as a solid block, eating away the margins of independent groups by first polarizing their borders into skeptical (1) and conformed (2) states.
